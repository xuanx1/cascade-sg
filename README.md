# CASCADE — Design Archive

A portfolio website showcasing CASCADE's design work across five disciplines:

- **Identities** — Brand and identity design
- **Spatial Planning** — Space and environment design
- **Industrial Design** — Product and object design
- **Data Visualisation** — Information and data-driven graphics
- **Intangibles** — Digital experiences, UX flows, and virtual work


## Project Structure

```
├── index.html              # Main landing page
├── loading.html            # Page loading screen
├── privacy-policy.html     # Privacy policy
├── 01identities/           # Identities category pages
├── 02spatial/              # Spatial Planning category pages
├── 03industrial/           # Industrial Design category pages
├── 04datavis/              # Data Visualisation category pages
├── 05intangible/           # Intangibles category pages
├── bg/                     # Background sketch images
├── css/                    # Stylesheets
├── fonts/                  # Self-hosted web fonts
├── images/                 # Shared images and favicons
└── js/                     # Scripts (GSAP, jQuery, Webflow, cursor)
```
